
# Al-Qur'an App

Al-Quran App makes people easier to read Quran everywhere and anytime.
## Installation
1. Clone this repository to your local machine:
```
git clone https://github.com/magistraapta/quran-app-swift
```
2. Open the file using Xcode
## Tech Stack
- SwiftUI
- Swift 5.5
- Xcode 14
